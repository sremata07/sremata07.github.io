## Now I know what HTML is

Seeing a .html in my web address bar was something I was always curious about, as I used to wonder what it meant and what it stood for. Well, now I know. Using HTML on its own is very barebones, it almost feels like you're trying to start a fire, but first you have to create the tools to start the fire in the first place. While relatively easy to learn, I felt like it was hard to to get the web page to look cool, and it felt like it was hard to make the HTML do certain things that I would have liked to see. 

## In comes the framework

Bootstrap 5 is the HTML framework we learned to use for this class, and if using raw HTML required making the tools on your own, using a framework like Bootstrap 5 is like being given all the tools and you just have to figure out how to use them effectively now. To me, it seems like frameworks exist to save time and hassle when working with HTML, and is a necessity if you want to be able to quickly style web pages. Now the only caveat is that if there is an issue with Bootstrap 5, you have to wait for the creators to deal with it, and this can be a problem if you have a bunch of web sites up using Bootstrap 5, and then there's some big vulnerability discovered within Bootstrap 5 that can pose a threat to your web pages. Of course the solution would be to create your own framework, but that means you have to also upkeep your own code and make sure it continues to work. So, like the tools analogy, these frameworks save time but also require tedious maintenance and attention in order to keep them working and available.

## To conclude,

Frameworks are incredible. I do not think I could handle coding in raw HTML without a framework, it just feels like I'm wasting time. While it is possible to accomplish what Bootstrap 5 does using just .css file and etc., why would you struggle to do that yourself when other people have already created the tools, and all you have to do is learn how to use them. Overall, I think that frameworks are a wonderful thing that can massively help with UI design, and of course, making sure that web page looks pretty.
